American Sign Language Alphabet to Speech

Translator for ASL to help create a real time conversation aid.
Trained on the kaggle dataset cited below, using google's mediapipe 
hand landmarking tools to preprocess frames and infer the alphabet.

Backend supported by Flask API with an HTML frontend that runs an application
locally on the user's device.


Dataset: https://www.kaggle.com/datasets/grassknoted/asl-alphabet/data
